# DF-Inicio_de_Sesión

##  Descripción del Problema
Diseñar un diagrama de flujo de un inicio de sesión donde el usuario pueda ingresar su usuario y contraseña. Si la contraseña es incorrecta, se muestra un mensaje de error y se tiene 2 intentos más. Si la contraseña es correcta, mostrar mensaje de bienvenida.
